The chat on OpenHospi is **end-to-end encrypted** (E2EE). This means only you and your conversation partner can read the messages. Not even we can.

<Callout type="info">
  Chat conversations are automatically created when a lister accepts a response to a listing. You
  don't need to set anything up.
</Callout>
