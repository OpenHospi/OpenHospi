De chat op OpenHospi is **end-to-end versleuteld** (E2EE). Dat betekent dat alleen jij en je gesprekspartner de berichten kunnen lezen — zelfs wij niet.

<Callout type="info">
  Chatgesprekken worden automatisch aangemaakt wanneer een aanbieder een reactie op een listing
  accepteert. Je hoeft zelf niets in te stellen.
</Callout>
